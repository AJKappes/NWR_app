Nutrient Waste Recycling Model

This model computes anaerobic digestion (AD) phosphorus output for lagoon application and energy production in kilowatt hours from user feedstock and manure inputs. The model also computes total revenue generated from tipping fees.

Enter input values below for computing AD phosphorus output (lbs) and recommended lbs/acre crop application. Energy production (kWh) and tipping fee revenue information is provided below phosphorus output and application recommendation.

{{form.blood}} Blood (gallons)
{{form.fish_by}} Fish Byproduct (gallons)
{{form.paper_pulp}} Paper Pulp (gallons)
{{form.ob}} Out-of-Date Beverages (gallons)
{{form.gt}} Grease Trap Waste (gallons)
{{form.manure}} Manure (gallons)
{{form.loc}} Type location as "West" for application west of the Cascades or "East" for application east of the Cascades
{{form.soil_test}} Current phosphorus soil test value
{{form.acres}} Number of crop acres for phosphorus application
